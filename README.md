# AI---powered-Emotion-Cam
Emotion Cam is a real-time emotion detection system that uses a webcam to capture facial expressions and analyze them using computer vision and deep learning techniques. The system is built with a React frontend for interactive user interface and a Python (Flask) backend for processing. It uses OpenCV to detect faces and DeepFace, powered by TensorFlow, to classify emotions such as happy, sad, angry, and neutral in real time. The application demonstrates seamless integration between frontend and backend, enabling fast and efficient emotion recognition with potential applications in smart education, user experience analysis, and human-computer interaction
